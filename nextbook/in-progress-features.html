Work In Progress Features

Here is a list of features which are planned to be implemented in future in no particular order.

  • In-page TOC indentation for sub headings.
  • Localization.
  • Sitemap generation on build.
  • Top progressbar as scroll happens.
  • Tabbed markdown content.
  • Folding side menu contents.
  • Fenced code: highlight lines without displaying line numbers.
  • Force hide in-page TOC menu.
  • Analytics with Splitbee.
  • Standalone chapters with no part information.
  • Cross-part navigation.
  • Fenced code diff support.
  • NPX template support.
  • Print styles.
  • Migrate to Next MDX Remote
  • Migrate to pure CSS or Tailwind
  • Multiple books support
  • Tag index page
  • Internationalisation
  • Global search. (Algolia?)
  • Live code edit & preview support.
  • Epub export.
